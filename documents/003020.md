# TypeScript 容易忽略的特性

## Basic Types

### unknown

``` javascript
let notSure: unknown = 4;
notSure = 'a string instead';
```

### Tuple（元组）

``` javascript
let x: [string, number] = ['hello', 10];
```

### enum

``` javascript
enum Color {
    Red = 1,
        Green,
        Blue,
}
let colorName: string = Color[2]; // colorName = 'Green'
```

### interface

#### readonly

``` javascript
interface Point {
    readonly x: number;
    readonly y: number;
}
```

#### ReadonlyArray

``` javascript
let ro: ReadonlyArray < number > = [1, 2, 3, 4];
```
